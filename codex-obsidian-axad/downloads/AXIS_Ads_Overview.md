# AXIS アド事業部 Overview

## 事業概要

| 項目 | 内容 |
|------|------|
| 事業内容 | アドアフィリエイト |
| 位置づけ | AXISホールディングスの主要事業。売上の大部分を占める |

## アドアフィリエイトとは

広告費を払って集客し、その流入をアフィリエイト成果につなげて差益を得るビジネスモデル。

```text
広告 -> 記事LP -> 本LP -> フォーム -> CV
```

| ステップ | 役割 |
|---------|------|
| 広告 | SNSや動画面で興味を引き、クリックさせる |
| 記事LP | 商品やサービスの魅力を記事風に伝える |
| 本LP | 公式ページで申込フォームへ誘導する |
| フォーム | ユーザーが情報を入力する |
| CV | 成果地点に到達する |

## 利益構造

```text
利益 = 成果報酬 - 広告費
```

広告費を先に投下し、成果報酬との差分を利益にする。ROAS、CPA、CV数、単価、広告費の関係を見る。

## 主な関係者

| 関係者 | 役割 |
|--------|------|
| クライアント | 商品・サービス提供、成果報酬・成果地点を設定 |
| ASP | クライアントと代理店を仲介し、成果計測・承認を担う |
| 代理店 | 広告を出稿し、成果を獲得する |
| 媒体 | Meta、TikTok、Google、X、LINEなど |
| ユーザー | 広告からLPへ進み、成果地点に到達する |

## AXIS Adsの特徴

| 項目 | 内容 |
|------|------|
| 案件タイプ | リード獲得が中心 |
| クライアント属性 | 店舗型ビジネスが多い |
| 主要媒体 | Meta / TikTok / Google / X / LINE |
| 主な成果地点 | 予約完了など |

## 強み

- 自社リスクで運用するため、利益最大化へのインセンティブが強い
- 高速PDCAでクリエイティブを大量に検証する
- 感覚ではなく数字で判断する

## 重要指標

| 指標 | 意味 | 計算式 |
|------|------|--------|
| ROAS | 広告費に対する売上 | 売上 ÷ 広告費 × 100% |
| CPA | 1件獲得にかかる広告費 | 広告費 ÷ CV数 |
| 利益 | 最終的に残る金額 | 売上 - 広告費 |
| CPC | 1クリックあたりの広告費 | 広告費 ÷ クリック数 |
| CTR | 表示に対するクリック率 | クリック数 ÷ 表示回数 |
| CVR | クリック後の成果率 | CV数 ÷ クリック数 |
| CPM | 1,000回表示あたりの広告費 | 広告費 ÷ 表示回数 × 1000 |

## クリエイティブ

アドアフィリエイトでは、クリエイティブが最重要要素の1つ。

```text
見出し -> ビジュアル -> 本文 -> LP遷移
```

勝ちクリエイティブが見つかったら、見出し、色、訴求、構成を変えて横展開する。

## 運用の基本サイクル

| タイミング | 主な作業 |
|-----------|----------|
| 朝 | 前日の成果確認、停止・予算調整 |
| 日中 | 新規クリエイティブ作成、出稿準備 |
| 夕方 | 分析、改善仮説、次の検証設計 |

## Codexに参照してほしい判断軸

- ROAS、CPA、CV数、広告費、利益を分けて見る
- クリエイティブ起点で改善余地を考える
- 数字の変化は、媒体、案件、訴求、LP、計測のどこに起因するかを分解する
- 事実、解釈、次に確認すべきことを分けて出す
