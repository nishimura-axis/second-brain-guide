## 講義の目的

1. マーケティングフレームワークを使った市場の捉え方を学ぶ
2. 競合メディア（代理店）の特徴を理解する
3. リサーチツールの正しい使い方を知る

---

## 前提：我々の職業とは

### マーケターである

- 「職業は何ですか」と聞かれたら**マーケター**と答える
- マーケターとは「商品やサービスを、それを欲している人に届ける」仕事
- インターネットを使ってやるのが**ウェブマーケター**

### 広告代理店である

- クライアントの商品・サービスを世に売るための広告を代理で受け持つ
- 広告を使って集客し、お金をもらう

### 意識すべきこと

- クリエイティブリサーチでユーザーのことを理解するのは大事
- ただし「どういう戦略で売上を上げようとして事業主が商品・サービスを作っているか」も理解する必要がある
- 商品・サービスの解像度を高く持った上でリサーチし、制作・運用に落とし込む

---

## 1. マーケティングフレームワーク

### 木口がよく使う3つのフレームワーク

| フレームワーク | 用途 |
|---------------|------|
| **3C分析** | 最も使う。案件単位・クリエイティブ単位でも使える |
| **SWOT分析** | 強み・弱み・機会・脅威の分析 |
| **STP分析** | ポジショニングを明らかにする |

### 3C分析とは

| 要素 | 英語 | 意味 |
|------|------|------|
| Customer | 市場・顧客 | ターゲットユーザー |
| Competitor | 競合 | 同じ市場の競争相手 |
| Company | 自社 | 自社の強み・弱み |

市場動向、顧客ニーズ、競合各社のシェア・推移、自社と競合の強み・弱みなどを分析し、自社の市場における立ち位置を把握する。

---

## 2. 3C分析の実践的な使い方

### 例：医療痩身（ディオ）の場合

**Step 1：競合ジャンルを洗い出す**

「痩せたい」という悩みを解決する手段の競合：

| ジャンル | 例 |
|----------|-----|
| 医療ダイエット | ウララクリニック |
| 痩身エステ | - |
| ジム | チョコザップ、ゴールドジム |
| 脂肪吸引 | - |
| 健康食品・サプリ | - |
| 自主トレ | - |

**Step 2：自社の優位性を明確にする**

- 案件に着手する前に「ディオは他の競合ジャンルと比べてどこに優位性があるのか」を明確にする
- これが**STP分析**（ポジショニングを明らかにする）

### ChatGPTの活用

競合リサーチに時間をかけるのはもったいない。ChatGPTを活用する。

**プロンプト例：**

1. 「痩せるための手段を教えて」
2. 手段が出てきたら「医療ダイエットとそれぞれを比較し、医療ダイエットの優位点を教えて」

→ 誰でもできる。すぐ落とし込める。

## 3. 施術メニューの理解

### なぜ施術メニューを知るべきか

1. **売り方の解像度が上がる** - 悩みに直接解決できる話ができる
2. **ユーザーリサーチのアンテナが高くなる** - N1リサーチで取りこぼしを防げる
3. **新しい訴求が生まれる** - 冒頭のアイデアの幅が広がる

### 例：TCBの場合

- 施術ごとにメニューが分かれている
- 施術について詳しくならないと構成が書けない
- 冒頭の軸も経験値がないと出てこない

### ChatGPTでの調査方法

**例：クマ再生の場合**

1. 「クマ再生注射とは」
2. 「どういう効果があるか」
3. 「注意点は何か」
4. 納得できなければ深掘り

→ 施術メニューや効果は**普遍**なので、一度知っておけばOK

---

## 4. 3C分析の2つの視点

### 視点1：クライアント商品の競合

- ディオ vs ウララクリニック vs 痩身エステ vs ジム etc.
- 商材ジャンル間での差分を測る

### 視点2：メディア（代理店）の競合

- アクシス vs ナハト vs KITEN etc.
- **同じ商材を扱っている競合メディアも詳しく調べる**

```
自社を知る（商材の強み）
     ↓
相手を知る（競合メディアの戦術）
     ↓
勝ち筋を計算する
```

---

## 5. 競合メディア（代理店）の特徴

### なぜ代理店ごとの特徴を知るべきか

- **会社の色・戦術はほとんど変わらない**
- どの会社も過去の成功体験を投下する（生存者バイアス）
- 代理店を知っていれば、リサーチ時に参考にすべきかどうか判断できる

### 主要代理店の特徴

| 代理店 | 特徴 |
|--------|------|
| **ナハト** | クリエイティブが強い。記事はあまり回収しない。アイデアベースで市場にないものを作る |
| **KITEN** | オーガニック起点でクリエイティブを作る。審査対策がうまい（Atom等で攻めている）。アクシスと仲が良い |
| **senjin** | アドアフィ事業を行っている競合会社。冒頭の外（バリエーション）をめっちゃ作る。審査対策がうまい。カット文字本編はsenjin発祥 |
| **ブリーチ** | 他商材からの横転。空いている枠を攻める。ブルーオーシャン戦略がうまい |
| **猿** | 記事が強い。クライアントROASが良い |
| **グリフィン** | 条件（単価）が良い傾向 |
| **ブレイブ** | リアクション冒頭、白背景＋音声BGMのパターン |

### クリエイティブ参考の優先順位

| 目的 | 参考にすべき代理店 |
|------|-------------------|
| クリエイティブ全般 | ナハト、KITEN、senjin（一部跳ねているもの） |
| 審査対策 | KITEN、senjin |
| 記事 | ブリーチ、猿、senjin |
| ブルーオーシャン戦略 | ブリーチ、グリフィン |

### 代理店の見極め方

| 方法 | 説明 |
|------|------|
| FBページ | 同じFBページを使っていれば同じ代理店 |
| 記事の共通要素 | 下部のボタン・予約フォームのステップ画像は変わらない |
| 音声・編集の特徴 | KITEN＝ゆうと君の妹、ブリーチ＝同じ声優を数年使用 |
| 過去案件 | FBページを下まで遡ると過去案件が見える |

## 6. 勝ち筋の考え方

### クリエイティブの勝ち筋（3軸）

1. **他社より高クオリティのクリエイティブを作る**
2. **他案件で取れているクリエイティブを横転する**
3. **オーガニックでバズっているものから市場にないものを作る**

### 戦略の立て方

- ブリーチとグリフィンしかいない市場 → **クリエイティブ力で勝てる**（後乗りでも勝てる）
- ナハトがいる市場 → **動画でぶち抜くのは大変**

### 各社の良い要素を掛け合わせる

**例：**
- senjinの冒頭（バリエーション多い）× KITENの審査対策本編
- → 新しいクリエイティブが生まれる

過去うちで売れたものも、代理店ベースで考えると「代理店と代理店の掛け合わせ」で作られている可能性がある。

## 7. リサーチツールの注意点

### 予算運用とアドアフィの違い

| 項目 | 予算運用 | アドアフィ |
|------|----------|-----------|
| 利益の発生 | 消化金額の20%が利益 | 売上 − 消化金額 = 利益 |
| 重視する指標 | **消化金額**（多いほど利益増） | **CPA**（安いほど利益増） |
| 目的 | 定常的な集客 | 拡大戦略 |

### なぜこれが重要か

- 予算運用は消化金額重視 → **再生回数が多くてもCPAが良いとは限らない**
- ツールで上位にいるからといって、そのクリエイティブが良いとは限らない

### 予算運用の見極め方

| 方法 | 説明 |
|------|------|
| 再生回数が綺麗な数字 | 1000単位で止まっている |
| 日々の消化金額が一定 | 予算が決まっているから |
| クリエイティブのクオリティが低いのに上位 | 予算で回しているだけ |

### 媒体ごとのCPMの違い

| 媒体 | CPM目安 | 特徴 |
|------|---------|------|
| Meta | 5,000〜10,000円 | 高め |
| TikTok | 1,000円程度 | 安い → 再生回数が伸びやすい |
| X | 安め | 再生回数が伸びやすい |

- TikTokやXは同じ金額でもインプレッションが多く出る
- ツールで上位に来やすいが、**TikTokだから取れているわけではない**

### 結論

- ツールを盲信しない
- 代理店ごとの特徴を頭に入れてリサーチする
- 上に書いたカオスマップにない代理店のクリエイティブは「弱小メディアか予算」と捉えてOK

---

## 8. まとめ

1. **マーケティングフレームワークを使う** - 3C、SWOT、STPを活用
2. **3Cの時に2つの視点を持つ**
   - クライアントの競合（商材ジャンルの優位性）
   - メディアの競合（どの代理店がいるか）
3. **代理店の特徴を当てはめて勝ち筋を計算する**
   - どこのポジションが取れるか
   - どういうクリエイティブを作れば勝てるか
