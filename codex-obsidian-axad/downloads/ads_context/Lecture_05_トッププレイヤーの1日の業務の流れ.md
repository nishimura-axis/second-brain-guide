## 1. Meta運用で利益を出す方法

**利益を出すのは2つだけ**
1. 運用キャンペーン数を増やす
2. 精査をする

この2つで利益が出る。

## 2. 1日の流れ

### 朝精査

朝精査はその日1日のROASだけでなく、翌日以降のキャンペーン継続・停止判断にも関わる重要な業務。

**精査の基準**
- 商材全体の当日の調子を見る（例：クマ再生全体で今日CVRが低いか）
- 商材に対しての基準値を設ける
- 案件全体の基準値・平均値を参考にする（調子いい期間をピックして平均数値で精査）
- キャンペーン自体のパワー（過去よく取れてたやつが最近不調なら期待値を追う）

**鹿島のやり方**
- キャンペーン数が多いので朝は結構シビアに切る
- 150キャンペーンで朝精査は約20分
- キャンペーンの「火」（期待値）は見ない、数値だけで判断
- 自分が決めた基準以下なら全部切る、それ以上なら一旦継続

**基準値を頭に入れる**
- 運用シート・分析シートを見て基準値を把握
- 動画と静止画でCPC・ROASの数値感は絶対変わる
- 新規商材でも数日で数値感を掴んで精度を上げる

### 昼

- 会社についたらまず精査
- CVついてるキャンペーンは朝は残すが、数値感悪かったら昼に停止
- **12時頃にはCVついてないキャンペーンは停止**
- どんなにCPCやMCPAが良くても1万円までしか見ない（大体12時頃に1万円ぐらい消化）
- 前日どんなに良くても、その日の数値で判断（感情ゼロ）
- 精査シートは30分〜1時間に1回見る（12時までは）

**理由**
- 平日の昼間ずっと精査してると制作にかける工数がなくなる
- 12時には取れてるやつだけにして制作に注力

### 翌日判断

**キャンペーン作り替えについて**
- セット作り替えはなし（ナハトもやってない、オンオフだけで対応）
- キャンペーン作り替えは数値が変わる感触がある
  - 例：CPCが100〜200円でめちゃいいけどMCV0で3日続いた → 作り替えたらCPC600円になって全然違う面に配信できた
- 判断：大体3日ぐらい見てキャンペーン作り替え、それでも2〜3日悪ければ完全停止

**施策単位で見る**
- 同じスクリプト・同じ見出しで年齢展開して全部ダメだったら、キャンペーン作り替えもせず全部完全停止
- キャンペーン1つ1つだけでなく、施策単位（トータル）で見る

**トータルROASと連続何日つかなかったかで判断**
- 例：ROAS200%で何十万出てるキャンペーンなら5日ぐらい利益出てなくても継続
- クリエイティブは1週間2週間で枯れると思ってない、1ヶ月は持つ
- いい面に会えば戦える

### 出稿

**0時から走らせることの重要性**
- Metaは0時からキャンペーンがちゃんと走らないと、ちゃんとした学習がかからない
- 出稿依頼が遅くなって3時〜5時に開始すると、7時に起きた時に消化1000円みたいなことになる
- そういうキャンペーンは1日無駄になり、トータルマイナスになりがち

**作り替えも同様**
- キャンペーン作り替えは遅くても8時〜9時には終わらせる
- 1日目が無駄になると、それが引きずって停止判断されやすくなる
- 運用キャンペーン数が減っていく原因

## 3. 運用知識

**木口がまとめた資料を週に1〜2回見る**
- 自分が何をやりきれてないか確認する時間を取る
- 詰め合わせ、ASCなど

**同じメディアIDを使う**
- クリエイティブの展開以外は基本的に同じメディアIDでやる
- 詰め合わせやASCをやる時も同じメディアIDを使う

## 4. ASC（Advantage+ Shopping Campaigns）

### ASCの実績

- 1月：ASCだけでROAS200%で470万利益
- エミナルの継続中キャンペーン：11月22日から走り続けて1キャンペーンで425万利益
- アリシアで7月〜9月に600万以上利益

**結論：ASCは通常キャンペーンより長期運用が可能**

### ASC化の条件

- ROASが130%以上
- 10CV以上ついている
- この条件を満たせば全部ぶっ込んでもいい

### ASCの設計

**基本的な考え方**
- 売れてるノンタゲのキャンペーンをただASCに変えるだけ
- 元のキャンペーンの組み合わせは変えない
- 広告を10個にする（足りなければ複製して10個にする）
- 同じメディアIDを使う

**構成例**
- 1キャンペーン・1広告セット・10広告（110）
- 元々151（1キャンペーン・5広告セット・1広告）なら、5個を掛けにして10個にする

**元々売れてなかったクリエイティブも入れる**
- ASC化する前は売れてなかったクリエイティブも、ASC化後に日によっては寄ってくるケースがある
- 死にクリエイティブだったものも全部突っ込む

### ASCの配信ロジック（仮説）

**通常キャンペーンの場合**
- 配信面を選んだ後、時間や日付が変わるたびに飛び飛びで面が移動する
- ピクセルで学習した情報の確度が薄いところにも移動してしまう
- 取れてたところから一気に移動してしまうのでブレやすい

**ASCの場合**
- 取れてるところを残しながら、ちょっとずつ近くのところを増やして新しいところを探す
- 自分たちの領域を徐々に広げていく
- だから長期で取れる

**媒体がASCを優遇する理由**
- 媒体推奨のキャンペーン設計なので絶対優遇される（TikTokのスマートプラスと一緒）
- 媒体がアップデートをかけ続けるのでそっちが良くなる
- 媒体側にとっては金を使いやすくなるメリットがある
- 魚群探知機をつけてやるから、もっと消化しろという構造

### ASCの初期設定・運用方法

**予算**
- 初期：3万円でスタート
- 上げ方：3万 → 5万 → 8万 → 12万 → 15万 → 20万
- 朝時点で1万消化で3〜4CVみたいにROAS合ってる時は1段すっ飛ばして上げる
- MAX20万まで（適正なCPMを維持できる上限）

**予算を下げるケース**
- ROAS合わない日が数日続いたら予算を3万まで戻す
- ネガティブなアクションを加えると数値改善するケースがある
- それで件数ついてくるケースは全然ある

**ASCの精査基準**
- 軌道に乗ってるキャンペーン（トータルでROAS合ってる）は基本オンオフだけ
- 5日連続赤でも全然運用する（トータルが合ってれば）
- 後日件数ついて予算上げて利益10万出して5日の赤を巻き返すケースは存在する
- 通常キャンペーンよりちょっと長めに見てあげる

**ASCのキャンペーン作り替え**
- するケース①：最初3日間でつかなかった時（通常キャンペーンと同じ）
- するケース②：予算上げて好調 → 取れなくなってネガティブアクション（予算3万に下げ） → それでも取れない → 作り替え

## 5. 通常キャンペーンの設計

**予算**
- 2万〜3万で組む
- 商材の波による（直近不調なら2万、普通に利益出るなら3万）
- 下限は商材の許容CPAの1.5倍ぐらいとされている

**ノンタゲの設計**
- 年齢・地域セグメントはできる限り広めに取る
- 例：医療脱毛のメインターゲット層が18〜39歳でも、18〜59で広く取った方がノンタゲ獲得できた経験がある

**11Nで出稿・検証する**
- 背景：配置（出面）によって結果が大きく変わるデータが出た
- オーディエンスネットワークに配信が寄りがち、オーディエンスネットワークはCPCがかなり安く出がち
- 5個セットで作ると、1つはオーディエンスネットワークに寄り、もう1つはインスタのストーリー・フィードに寄るなど、クリエイティブではなく配置でCPCが変わる
- 正確な検証結果を得づらい
- **全部同じ面にするために11Nにする**

**1キャンペーンのクリエイティブ数**
- 5〜10ぐらいにとどめる
- 150個のクリエイティブを1キャンペーン3万予算で検証するのはもったいない
- 素材×冒頭のパターンで軸を作って複数キャンペーン回す
- 1個1個のクリエイティブを大切にする

## 6. 詰め合わせ

**詰め合わせとは**
- 好調なキャンペーンのセット全部を丸ごと1つのキャンペーンに統合すること

**効果的な詰め合わせ**
- いろんな施策（年齢展開・地域展開・検証クリエイティブなど）、セグメントの違いがあるものを1つにまとめた方が数値がかなり良かった
- AIに対していろんな手札を持たせておくと、市況の変化などに対して適したものを切ってくれる

**詰め合わせのやり方**
- ノンタゲ・年齢・地域を全部詰め合わせる（70セットぐらいのキャンペーンで利益出した成功体験あり）
- 年齢だけ、地域だけなど偏った詰め合わせはうまくいきにくい
- 結果ついてないセットも入れる（キャンペーンとして取れてればその施策自体は悪くない、たまたまその時点で最悪だっただけ）

---

## 7. 除外（地域除外）

**除外とは**
- 地域展開で複数セットを作った時に、消化が寄ってる地域を除外して別キャンペーンを立てる施策

**やり方**
- 例：TCBで105院を10キャンペーンぐらいに地域分け（1キャンペーン10セットぐらい）
- ボリューム出てるところ（消化が寄ってる地域）を抜く
- 取れてるキャンペーンでも、消化が寄ってる地域が邪魔してる可能性がある
- 他の地域で取れる可能性がある

**考え方**
- いいやつに5万とか寄ってくれた方が取れる
- 悪いのに消化が増えるのを防ぐ
- 施策自体がトータルで利益出てればやってもいい

---

## 8. 全年齢詰め合わせ（ナハトから持ち帰った施策）

**概要**
- 全年齢（18〜65歳）を全部1キャンペーンにして出す
- 予算10万とかでスタート
- 構成例：1キャンペーン・140セット（年齢別クリエイティブ全部入れる）

**実績**
- 1月18日から動いてROAS180%で170万利益

**仮説**
- AIが能動的にいい年齢などに寄せてくれている

## 用語解説

| 用語 | 説明 |
|------|------|
| ASC | Advantage+ Shopping Campaigns。Meta広告の自動最適化キャンペーン |
| ノンタゲ | セグメントを絞らない配信 |
| 詰め合わせ | 複数のキャンペーンを1つに統合すること |
| 110 / 11N | キャンペーン・広告セット・広告の構成数（例：1キャンペーン・1広告セット・10広告） |
| メディアID | クリエイティブを媒体にアップロードしたときに割り振られる固有のID |
| 年齢展開 / 地域展開 | ノンタゲで当たったCRに年齢や地域の訴求を追加して、セグメントを細分化して利益を最大化する施策 |
| オーディエンスネットワーク | Meta広告のパートナーアプリへの配信面 |
| 除外 | 配信が集中している地域を除外して別キャンペーンを立てる施策 |
| セット予算 / キャンペ予算 | 広告セット単位の予算 vs キャンペーン単位の予算 |
