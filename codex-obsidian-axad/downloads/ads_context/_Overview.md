## アドアフィリエイトとは

**「広告を出稿してアフィリエイト成果を獲得するモデル」**

広告費を払って集客し、その流入をアフィリエイト成果につなげて差益を得るビジネスモデル。

### 関係者

| 関係者 | 役割 |
|--------|------|
| **クライアント（広告主）** | 商品/サービス提供、成果報酬・成果地点を設定 |
| **ASP** | クライアントと代理店を仲介、成果計測・承認 |
| **代理店（AXIS-Ads）** | 広告を出稿して集客し、成果を獲得する |
| **媒体** | Meta, TikTok, Google(YT), X, LINE |
| **ユーザー** | 広告→LP→成果地点と流れる |

※成果地点はクライアントが決める（交渉は可能）

### ユーザー導線

```
【広告】→【記事LP】→【本LP】→【フォーム】→【CV（成果地点）】
```

| ステップ | 役割 |
|---------|------|
| 広告 | SNSのフィードに表示。興味を引いてクリックさせる |
| 記事LP | 商品の魅力を「記事風」に伝える。読み進めさせる |
| 本LP | クライアントの公式ページ。申込フォームへ誘導 |
| フォーム | 個人情報を入力して申込完了 |
| CV | 成果地点に到達（予約完了など） |

### 利益構造

```
利益 = 成果報酬 − 広告費
```

**例：**
- 成果報酬：15,000円/件
- 広告費：10,000円で1件獲得
- **利益：5,000円**

## AXIS-Adsの特徴

| 項目 | 内容 |
|------|------|
| **案件タイプ** | リード獲得メイン |
| **クライアント属性** | 店舗型が大半（クリニック系：美容整形、医療痩身、医療脱毛等） |
| **主要媒体** | Meta / TikTok / Google(YT) / X / LINE |
| **主な成果地点** | 予約完了 |

### 強み

| 強み | 説明 |
|------|------|
| **自社リスクで運用** | 本気で利益を追求するインセンティブ |
| **高速PDCA** | 1日で数十パターンのCRをテスト |
| **データドリブン** | 感覚ではなく数字で判断 |

### メリット

| メリット | 説明 |
|----------|------|
| **即時性** | SEOで上位化を待たず、広告で即座にアクセス獲得可能 |
| **スケーラビリティ** | 広告費を増やせば流入も増やせる |

### リスク

| リスク | 説明 |
|--------|------|
| **先行投資** | 広告費が先にかかり、成果が出るまで赤字リスク |
| **運用スキル依存** | 広告運用の知見がROIに直結 |
| **規約リスク** | 媒体の規約違反でアカウント停止リスク |

## 主要媒体

| 媒体 | 特徴 |
|------|------|
| **Meta（Instagram/Facebook）** | 最も出稿量が多い。30〜50代女性に強い |
| **TikTok** | 若年層向け。動画クリエイティブが中心 |
| **Google(YouTube)** | 動画広告。検索連動も可能 |
| **X（旧Twitter）** | テキスト+画像。拡散力がある |
| **LINE** | 日本国内に強い。友だち追加→配信の導線 |

### アカウント構造（Metaの例）

```
ビジネスマネージャ（BM）
  └─ 広告アカウント
       └─ ピクセル（計測タグ）
            └─ キャンペーン
                 └─ 広告セット（ターゲティング・予算）
                      └─ 広告（クリエイティブ）
```

**ピクセル** = 「誰がCVしたか」を媒体に伝えるタグ。これがないと最適化が効かない。

## 重要指標

### 収益性の指標

| 指標 | 意味 | 計算式 |
|------|------|--------|
| **ROAS** | 広告費に対する売上 | 売上 ÷ 広告費 × 100% |
| **CPA** | 1件獲得にかかる広告費 | 広告費 ÷ CV数 |
| **利益** | 最終的に手元に残るお金 | 売上 − 広告費 |

**ROASの目安：**
- ROAS 100% = トントン（利益ゼロ）
- ROAS 150% = 広告費1万円で1.5万円売上 → 利益5,000円

### 広告効率の指標

| 指標 | 意味 |
|------|------|
| **CPC** | 1クリックあたりの広告費 |
| **CTR** | 広告を見た人のうち、クリックした割合 |
| **CVR** | クリックした人のうち、購入した割合 |
| **CPM** | 1,000回表示あたりの広告費 |

### 要素分解

```
利益 = 売上 − 広告費
     = (CV数 × 単価) − 広告費
     = (クリック数 × CVR × 単価) − (インプレッション × CPM)
```

どの変数を動かせば利益が最大化するか？を考える。

## クリエイティブ（CR）

アドアフィで最も重要な要素。

### 種類

| 種類 | 特徴 |
|------|------|
| **静止画** | 1枚画像。制作コスト低い。大量に検証しやすい |
| **動画** | 情報量多い。TikTok/YouTubeでは必須 |

### 静止画CRの構造

```
┌─────────────────────┐
│  見出し（キャッチコピー）  │ ← 最重要。これでスクロールを止める
├─────────────────────┤
│   ビジュアル（画像）    │ ← 商品・人物・Before/After等
├─────────────────────┤
│   スクリプト（本文）    │ ← 詳細説明。読ませる
└─────────────────────┘
```

**勝ちCRを見つけたら「横展開」** = 見出しだけ変える、色だけ変える等で派生を作る

## 運用業務の流れ

### 朝：精査
- 前日の広告パフォーマンスを確認
- ROASが低い広告は停止
- ROASが高い広告は予算追加（ブースト）

### 日中：出稿準備
- 新しいクリエイティブを作成
- 広告セット・キャンペーンを設計
- 0時に配信開始（ピクセル学習のため）

### 夕方：分析・改善
- どの要素が効いているか分析
- 次のテスト仮説を立てる

## 競合分析

### 予算運用 vs アドアフィの違い

| 項目 | 予算運用（代理店） | アドアフィ |
|------|------------------|-----------|
| リスク | クライアント持ち | 自社持ち |
| 利益構造 | 手数料（広告費の20%等） | 成果報酬 − 広告費 |
| インセンティブ | 広告費を増やしたい | 利益を最大化したい |

### 競合代理店の見極め方
- Meta広告ライブラリで競合の広告を確認
- FBページから「どの代理店が運用しているか」を特定
- 勝ちCRのパターンを分析

## 用語定義

| 用語 | 意味 |
|------|------|
| **媒体** | 広告配信プラットフォーム（Meta, TikTok, YT等） |
| **代理店** | 広告を出稿して集客し、成果を獲得する側（AXIS-Ads） |
| **チャネル拡大** | 既存以外の新しい仕組みを導入して売上を発生させること。例：新媒体への出稿開拓、新しい成果地点・導線の導入 |
| **センターピン** | 今、最もインパクトがある改善ポイント |
